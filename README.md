# 🔥 Wildfire Detection Using Deep Learning

A Deep Learning project for binary image classification to detect **Fire** and **No Fire** using a Custom CNN and transfer learning models including **VGG16**, **ResNet50**, and **MobileNetV2**. The project compares model performance using TensorFlow/Keras to identify the most effective architecture for wildfire detection.

## 📌 Features

- Binary image classification (Fire / No Fire)
- Data preprocessing and augmentation
- Custom CNN implementation
- Transfer Learning with:
  - VGG16
  - ResNet50
  - MobileNetV2
- Model training and evaluation
- Performance comparison using validation accuracy

## 🛠️ Tech Stack

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Scikit-learn
- OpenCV

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/gazanfar-ali/Wildfire-Detection-Using-Deep-Learning.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the notebook

```bash
jupyter notebook notebook/Wildfire_Detection.ipynb
```

## 📊 Models Evaluated

- Custom CNN
- VGG16
- ResNet50
- MobileNetV2

## 📈 Results

The models were evaluated on validation accuracy and compared to determine the best-performing architecture for wildfire image classification. **MobileNetV2** achieved the highest validation accuracy among the evaluated models.
